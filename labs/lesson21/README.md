## Задачи
- Создание сети и настройка основных параметров устройства
- Настройка и проверка двух серверов DHCPv4 на R1

## Создание сети и настройка основных параметров устройства
### Схема адресации
Разобьём сеть 192.168.1.0/24 на несколько более мелких:
- Подсеть для клиентов **R1** (минимум 58 хостов): 192.168.1.0/26 (192.168.1.1-62)
- Подсеть управления (минимум 28 хостов): 192.168.1.64/27 (192.168.1.65-94)
- Подсеть для клиентов **R2** (минимум 12 хостов): 192.168.1.96/28 (192.168.1.97-110)

### Базовая настройка маршрутизаторов
Выполним базовую настройку на примере маршрутизатора **R1**
